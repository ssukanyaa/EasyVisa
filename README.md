# 🛂 Easy Visa – Predicting Visa Approvals

This project uses classification models to predict visa application outcomes (approved or denied) based on applicant features.

## 🧠 Objective
Build and compare machine learning models to classify visa applications as approved or denied, aiming to assist with decision automation.

## 📊 Dataset
- ⚠️ Dataset not included due to usage restrictions.

## 🔍 Workflow
- Data cleaning and preprocessing
- EDA
- Feature selection and correlation analysis
- Model training & Hyperparameter tuning
- Conclusion and Recommendations

## 📈 Evaluation
- Metrics: F1-score
- Best performing model: `XGBoost` with ~82% F1-score.
